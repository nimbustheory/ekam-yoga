# Ekam Yoga — Member Loyalty App Demo

8x Best of Bozeman winner. Downtown Bozeman's home for movement, community, and transformation.

## Quick Start
```bash
npm install
npm run dev
```

## Deploy to Vercel
```bash
npm run build
# Upload dist/ to Vercel, or connect your repo
```

## Structure
- `src/main.jsx` — Entry point, renders DemoWrapper
- `src/DemoWrapper.jsx` — Sales presentation shell (three-column layout)
- `src/demo.config.js` — Per-studio wrapper configuration
- `src/App.jsx` — Full loyalty app (consumer + admin)
